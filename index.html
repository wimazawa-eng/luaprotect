// Features

Heavy Obfuscation

Control-flow flattening, VM encryption, constant masking—pack your logic into an unreadable binary chunk.

License Engine

Seat-locked, game-locked, time-locked keys. Revoke or renew in real time from the terminal dashboard.

Secure Loader

Inject nothing. Load obfuscated modules only inside your own experiences via a signed require pipeline.

Analytics

Live telemetry: execution count, license violations, IP geolocation, instant email alerts.

// Quickstart

$ npm i -g luaprotect-cli
$ luaprotect login
$ luaprotect obfuscate script.lua --output script.luac
$ luaprotect license create --game 123456789 --expire 30d
$ luaprotect deploy script.luac
